# User Management

## Управление пользователями
- Администратор должен иметь возможность создавать новых пользователей
- Администратор должен иметь возможность редактировать профиль пользователя
- Администратор должен иметь возможность деактивировать/активировать пользователя
- Администратор должен иметь возможность назначать роль (Администратор/Оператор)
- Система должна отображать список всех пользователей с пагинацией

## Поиск и фильтрация
- Поиск пользователей по логину, имени, email
- Фильтрация по роли и статусу (активен/неактивен)
- Сортировка по дате создания, последнему входу, логину

## Действия с пользователями
- Возможность массового импорта пользователей из CSV
- Возможность экспорта списка пользователей в CSV/Excel
- Подтверждение действия при удалении пользователя
- Возможность временной блокировки пользователя
