# Dashboard

## Обзор
- Система должна отображать сводку по активным сессиям
- Система должна отображать количество онлайн-пользователей
- Система должна отображать график активности за последние 24 часа
- Система должна отображать статус подключенных устройств
- Система должна отображать последние события авторизации

## Виджеты
- Виджет "Активные сессии" с возможностью фильтрации по роли
- Виджет "Статус устройств" с цветовой индикацией (зелёный/жёлтый/красный)
- Виджет "Последние ошибки" с временной меткой
- Виджет "Статистика входов" за сегодня/неделю/месяц

## Обновление данных
- Данные на дашборде должны обновляться автоматически каждые 30 секунд
- Пользователь должен иметь возможность принудительно обновить данные
- При ошибке загрузки данных показывать заглушку с кнопкой повтора
