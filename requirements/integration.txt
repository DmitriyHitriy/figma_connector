# API Integration

## REST API
- Система должна предоставлять REST API для внешних интеграций
- API должен поддерживать аутентификацию по API-ключу
- API должен поддерживать аутентификацию по JWT-токену
- API должен возвращать данные в формате JSON
- API должен поддерживать пагинацию и фильтрацию

## Эндпоинты
- GET /api/users — список пользователей
- GET /api/users/{id} — информация о пользователе
- GET /api/devices — список устройств
- GET /api/devices/{id} — информация об устройстве
- GET /api/events — список событий
- GET /api/sessions — список активных сессий

## Документация
- API должен иметь интерактивную документацию (Swagger/OpenAPI)
- Документация должна содержать примеры запросов и ответов
- Документация должна быть доступна по адресу /api/docs
