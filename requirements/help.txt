# Help & Support

## Справка
- Система должна предоставлять встроенную справочную систему
- Справка должна быть доступна из любого раздела интерфейса
- Справка должна содержать описание всех функций системы
- Справка должна содержать инструкции по типовым операциям

## Поиск по справке
- Полнотекстовый поиск по справочным материалам
- Группировка результатов по категориям
- Отображение релевантности результатов поиска

## Контакты поддержки
- Отображение контактных данных службы поддержки
- Возможность отправки запроса в поддержку из интерфейса
- Отображение информации о версии программного обеспечения
- Отображение информации о лицензии
